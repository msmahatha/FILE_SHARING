# 📁 File Sharing App

A simple and secure file sharing web application built for fast and efficient file uploads and downloads.

🔗 **Live Site:** [filesharing007.netlify.app](https://filesharing007.netlify.app)

## 🚀 Features

- 📤 Upload and share files instantly
- 🔒 Secure and private file handling
- 🌐 Fully responsive UI
- ⚡ Built with modern web technologies

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Hosting: Netlify

## 📦 How to Use

1. Visit the live site: [filesharing007.netlify.app](https://filesharing007.netlify.app)
2. Click **Upload**, select your file.
3. Share the generated download link with anyone.
